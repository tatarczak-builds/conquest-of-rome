---------------------------------------------------
            Conquest of Rome

I created this game as part of my studies. It is currently available in my native language (Polish), though I might translate it into English as well. Apparently, it resembles the game *Swords and Sandals*, although I have never played it myself.


## Features
- choice of gladiator's class
- combat system
- experience and level progression
- random opponents
- item shop with level requirements
- equipment
- multiple locations
- multiple endings


# Technologies
- Python 
    - Classes
    - Inheritance
    - Modules


## Installation
 1. Clone the repository
 2. Run silnik.py
 3. Game runs in terminal
